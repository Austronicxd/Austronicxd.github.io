# Austronic Multiverse

Sitio oficial publicado mediante GitHub Pages.

## Agregar imágenes

Crea una carpeta `assets` y sube `hero.jpg`, `ogi.png`, `taxes.png`, `sunlor.png` y `mady.png`. Después agrega las imágenes como fondos en `styles.css`. La página funciona con marcadores mientras preparas los artes.

## Seguridad

No publiques correos personales, contraseñas, tokens ni información de pagos en este repositorio público.
